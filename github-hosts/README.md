# Github hosts 

<br />

写一个脚本，适合于Linux/Windows中运行，功能是当前主机可以访问github.com的代理脚本，代理主机搜索延迟最低的10个，每一小时更新一下代理主机IP地址池，保存到对应的文件夹中文件夹用当天日期命名，文件名用当天的日期+小时，是否更换代理主机代理池由用户决定。

1. 获取当前系统的信息，参考v2rayN-linux-64.deb，v2rayN-windows-64-desktop.zip格式
2. 搜索可以访问github.com 网站的代理主机IP地址，测试连通性，保存到文件中
   1. 第一列是IP
   2. 第二列是IP对应的域名
   3. 第三列是延迟（ms)
3. 编写适合当前CPU，OS的脚本
   1. 在命令行中可以运行的，多个步骤选择最终运行该程序
   2. <br />
4. 打法
